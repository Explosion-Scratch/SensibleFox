SensibleFox

An opinionated, zero-config Firefox for macOS.

This installer will download Firefox {{FF_VERSION}} and configure it with the SensibleFox defaults:

  • uBlock Origin with default filters + ClearURLs
  • Quad9 DNS over HTTPS
  • ~800 privacy and usability prefs from Betterfox
  • Clean tab bar, context menus, and chrome
  • Telemetry, Pocket, and onboarding removed
Download~{{FF_SIZE_MB}} MB
Installed~{{FF_INSTALLED_MB}} MB

A separate progress window will appear during the download. Existing Firefox installs at /Applications/Firefox.app will be replaced.
